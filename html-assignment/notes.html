5 HTML TAGS

Description List

The description list starts with an opening <dl> tag and ends with a closing </dl> tag.

Inside the opening <dl> and closing </dl> tags, we add the information using <dt> for the term and <dd> for the description.

Ordered List

Based on the name, ordered lists are used when the order of items in the list is important.

The ordered list starts with an opening <ol> tag and ends with a closing </ol> tag.

Unordered List

Unordered lists are used when the order of items in the list is not important.

The unordered list starts with an opening <ul> tag and ends with a closing </ul> tag.

Body

<body></body> is the main body of the document, which is what we see in our browsers.

All the content of the web page must be added inside the opening <body> and closing </body> tags.

Navigation

<nav></nav> represents the navigation of the page. This will contain the links to other pages of the website.
